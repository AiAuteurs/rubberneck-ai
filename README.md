# Rubberneck.ai

## Deploy to Vercel

1. Download this folder
2. Go to vercel.com → New Project → Import from your computer (or push to GitHub first)
3. Deploy — that's it

## To change today's featured site

In `public/index.html`, find this line:

```
<iframe src="/api/proxy?url=https://www.archivebuttons.com/"></iframe>
```

Change the URL to whatever site you want to feature. Redeploy.
